# AI-Powered Network Intrusion Detection Dashboard

## Overview

This project is a cybersecurity-focused machine learning application that analyzes network traffic and identifies potentially malicious activity.

The goal is to simulate part of a Security Operations Center (SOC) workflow by detecting suspicious traffic patterns and displaying security insights through an interactive dashboard.

This project is being developed as part of my cybersecurity portfolio to strengthen my skills in:

- Python
- Cybersecurity
- Machine Learning
- Network Analysis
- Security Monitoring
- Git/GitHub Workflows

---

## Project Goals

### Phase 1: Foundation

- Set up development environment
- Create project architecture
- Load and analyze network traffic datasets
- Build baseline dashboard

### Phase 2: Machine Learning

- Train intrusion detection models
- Evaluate classification performance
- Detect malicious traffic patterns
- Visualize security metrics

### Phase 3: Advanced Features

- Integrate CICIDS2017 dataset
- Improve model accuracy
- Add threat analytics
- Create SOC-style reporting views
